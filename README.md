# FishingBucket

FishingBucket is a Fluxer bot that proxies messages from users. For more information, please see ./data/help.md and ./data/explain.md

## Setup

Setup this project by installing Python 3.14 at minimum then install the dependencies listed in ./pyproject.toml.

Then, cd into the ./scripts/ directory and run each file.

Afterward, create a configuration JSON file. Follow the schema defined in ./config.schema.json.

Finally, run the bot by executing ./main.py with the argument being the path to the configuration file.
